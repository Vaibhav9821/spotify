Spotify Clone Web Player

This project is a Spotify-inspired web music player built using HTML, CSS, and JavaScript. It dynamically loads songs from folders, allowing users to browse albums, play tracks, control playback, adjust volume, seek through songs, and navigate playlists with a smooth and interactive UI.
 Features:
Album-based playlist system using folder structure
Play / Pause / Next / Previous controls

 How It Works

Songs are automatically loaded from local folders using JavaScript. Clicking an album loads its songs into the playlist, and clicking a song plays it instantly. All playback controls are handled using a single Audio object for smooth performance.
